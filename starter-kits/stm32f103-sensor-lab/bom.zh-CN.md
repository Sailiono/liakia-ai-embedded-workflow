# 物料清单

## 推荐基础套件

| 物料 | 推荐 | 备注 |
|---|---|---|
| MCU 板 | STM32F103C8T6 Blue Pill 类开发板 | 推荐选择芯片丝印清楚、3.3V 稳压正常的板 |
| 调试器 | ST-LINK/V2 或兼容调试器 | 用于 SWD 烧录和只读寄存器探查 |
| 串口模块 | USB-TTL，3.3V 电平 | 不要使用 5V TTL 直连 F103 引脚 |
| 传感器 | BMP280 I2C 模块 | 推荐选择带 3.3V 供电和 I2C 上拉的模块 |
| 电阻 | 4.7k x2 | 如果 BMP280 模块没有 I2C 上拉，用作 SDA/SCL 上拉 |
| 线材 | 杜邦线若干 | SWD、UART、I2C、GND 共地 |
| 可选工具 | 8 通道逻辑分析仪 | 用于观察 I2C / UART 时序和高级故障 |

## 不建议第一版使用的硬件

| 硬件 | 原因 |
|---|---|
| DHT11 / DHT22 | 单总线时序教学价值有，但自动化和寄存器证据链不够清晰 |
| DS18B20 | 适合温度采集，但协议和 MCU 外设关联较弱 |
| OLED 屏 | 视觉效果好，但会增加 I2C 设备数量，第一版容易干扰问题定位 |
| 多个传感器同时接入 | 对新手入口不友好，先把一个传感器闭环打穿 |
