台架接线

先把接线搭清楚，后面的失败才有意义。

主视觉用简化接线图而不是真实照片，避免线束外观差异干扰。关键是：SWD 用来烧录，USART1 用来看 shell 和日志，I2C1 用来接 BMP280，GND 必须共地。
接线检查表
  • SWDST-LINK -> SWDIO / SWCLK / GND
  • UARTUSB-UART -> PA9 / PA10 / GND
  • I2CBMP280 -> PB6 SCL / PB7 SDA
  • PowerBMP280 使用 3V3，所有模块共地

人工排查 vs AI 辅助

先自己排查，再让 AI 用同一份证据排查。

这个对比不是为了证明 AI 永远更准，而是让你直观看到：AI 能把搜索范围变窄，工程师仍然负责确认修复。
人工路径
  1. 看串口日志
  2. 怀疑接线或地址
  3. 检查 I2C 和 chip ID
  4. 排除硬件断线
  5. 继续定位补偿数据
目标：训练你自己的证据阅读能力。
AI 辅助路径
  1. 读取 ai_prompt.md
  2. 引用 chip_id PASS
  3. 排除传感器完全失联
  4. 聚焦 raw calibration / compensation
  5. 给出最小修复范围
目标：缩小排查范围，由工程师确认。
