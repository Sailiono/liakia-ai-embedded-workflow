the learning loop

先人工排查，再让 AI 排查，最后看答案。

1. 人工优先观察 FAIL gate、串口输出和 manifest，记录现象、假设、排除项和耗时。
2. AI 第二把同一份 evidence 交给 AI，要求它只能基于日志、gate result 和 raw value 推理。
3. 答案最后完成对比后再打开 ANSWER，用来校验你的诊断质量。
