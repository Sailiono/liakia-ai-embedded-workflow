how it works

dpiny-RTK is the demo case. Liakia is the reusable delivery workflow.

01

Build

CMake, Ninja, CubeCLT, and Arm GCC are invoked from scripts rather than a manual IDE session.

cmake --build --preset Debug
02

Flash

STM32CubeProgrammer performs SWD download, verify, and reset with logs preserved.

SWD / verify / reset
03

Test Gates

Serial shell, RTCM parser, USB CDC reset recovery, and register probes become repeatable gates.

shell / CRC / CDC / SWD probe
04

Evidence

Every run can emit a manifest, summaries, command transcripts, and firmware artifact records.

manifest.json / logs / handoff
