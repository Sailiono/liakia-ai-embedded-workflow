roi model

ROI 不是只讲 API 便宜，而是讲周期、风险和交接歧义被压缩。

traditional estimate

15-25 人天

CMake / CubeCLT 迁移0.5-1 天
USB CDC Shell 框架1-2 天
RTCM 转发和验证1-2 天
自动化和证据归档3-5 天
集成调试 buffer3-8 天
ai-assisted compression

3 人天 + 约 ¥10 API 消耗

实现驱动骨架、Shell 命令、配置结构
反馈编译 warning/error 快速分析
测试runner、parser 和 gate 复用
归档manifest、summary 和 handoff 自动沉淀
guardrails

AI 不替代工程判断

  • 原理图、供电、隔离、安全边界仍需人工审核。
  • EMC/ESD、协议边界、量产风险不能靠模型拍板。
  • 最终代码 review 和验收签字保留在人审闭环内。
